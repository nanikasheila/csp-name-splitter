# 用語集

**最終更新**: 2026年3月1日

---

## ドメイン用語

| 用語 | 定義 |
| --- | --- |
| **俯瞰ネーム** | 複数ページ分のネーム（漫画の下書き）を1枚の大きな画像にまとめたもの。本ツールの主入力 |
| **ネーム** | 漫画制作におけるコマ割り・セリフ配置の下書き段階 |
| **グリッド** | 俯瞰ネーム画像をページ単位に区切るための行列配置。例: 4行×4列 = 16ページ |
| **セル** | グリッドの1マス。1つのページに対応する矩形領域 |
| **マージン** | 俯瞰ネーム画像の外縁とグリッド領域の間の余白。上下左右を個別に設定可能 |
| **ガッター** | グリッドのセル間の余白（ページとページの間のスペース） |
| **読み順 (order)** | ページ番号を割り当てる方向。`rtl_ttb`（右→左、上→下）は日本の漫画標準 |
| **テンプレート** | ページレイアウトの雛形画像。罫線・枠線・背景などの要素で構成される |
| **フラット (flat)** | レイヤーを統合した単一画像。PSDの全レイヤーを1枚に結合した状態 |
| **マージルール** | PSD レイヤーのフィルタリング・グループ化の規則 |

## ソフトウェア略語

| 略語 | 正式名称 | 説明 |
| --- | --- | --- |
| **CSP** | CLIP STUDIO PAINT | イラスト・漫画制作ソフトウェア |
| **CSP EX** | CLIP STUDIO PAINT EX | 複数ページ管理機能を持つ上位エディション |
| **PSD** | Photoshop Document | Adobe Photoshop のファイル形式。レイヤー情報を保持 |
| **DPI** | Dots Per Inch | 解像度の単位。印刷品質に影響（300dpi: 標準、600dpi: 高品質） |

## 設定ファイル用語

| 用語 | 説明 |
| --- | --- |
| **plan.yaml / plan.json** | 分割設定ファイル。画像ごとにグリッド・マージン・出力設定を定義 |
| **GridConfig** | グリッド設定の dataclass。行列数、マージン、ガッター、DPI、ページサイズ等を保持 |
| **SplitConfig** | 分割設定全体の dataclass。GridConfig + MergeConfig + OutputConfig を統合 |
| **margin\_unit** | マージン値の単位（`px` または `mm`）。DPI と組み合わせて変換 |
| **page\_size\_name** | ページサイズのプリセット名。`A4`, `B5`, `Custom` 等 |

## GUI 固有用語

| 用語 | 説明 |
| --- | --- |
| **プレビュー** | 分割結果の事前確認。ページ番号オーバーレイ付きのサムネイル表示 |
| **キャンバスサイズ** | テンプレート生成時の出力画像サイズ（ページサイズ + マージン） |
| **Mixin** | GUI ハンドラ/ウィジェットの機能分割パターン。多重継承で統合される |

## 読み順パターン

| コード | 名称 | 用途 |
| --- | --- | --- |
| `rtl_ttb` | 右→左、上→下 | 日本の漫画（標準） |
| `ltr_ttb` | 左→右、上→下 | 西洋の漫画・コミック |
